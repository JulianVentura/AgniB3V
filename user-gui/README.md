# User GUI



### Installation

Install Python 3.10.  The recommended way is  [Pyenv](https://github.com/pyenv/pyenv)

```bash
pyenv install 3.10 -s
pyenv local 3.10
```
Install requirements:

```bash
pip install -r requirements.txt
```



### Execution

**Open user GUI**

```bash
python3 main.py
```
